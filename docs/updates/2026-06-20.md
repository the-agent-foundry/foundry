# 2026.06.20 update notes

This update batch expands Agent Foundry's founder-agent pattern library and adds the repo-level update-notification workflow.

## What changed

- Added/expanded founder-agent patterns for finance control, revenue/GTM, company context, meeting prep, Question Storm, and Auto-buildroom governance.
- Added example artifacts that make the patterns easier for another founder's agent to inspect and adapt.
- Added a release-notification workflow: GitHub Releases, `CHANGELOG.md`, `MIGRATION.md`, and `docs/release-process.md`.
- Added README instructions for subscribing to future updates.

## Who should care

- Founders using this repo as an agent operating-system pattern catalog.
- Agents reading the repo to recommend which patterns to build next.
- Maintainers of forks who want a clean way to track new uploads.

## Action needed

- Pull latest `main` after the update lands.
- Review `CHANGELOG.md` for the durable change history.
- Subscribe to GitHub Releases if you want future update notifications.
- Check `MIGRATION.md` before copying changes into a fork.

## Breaking changes

None.

## Safety posture

This batch is documentation/pattern oriented. Public-release safety still depends on the normal repo gates in `PUBLICATION_CHECKLIST.md` and `SECURITY.md`.
