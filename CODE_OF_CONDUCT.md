# Code of Conduct

This is a small, trusted group of founders sharing what works. Act like it.

## The short version

- Be useful. Share patterns, not noise.
- Be generous. Someone helped you get here. Pay it forward.
- Be honest. If something is a hack, say so. If you are guessing, say so.
- Be respectful. Disagree with the idea, not the person.
- Protect each other. Never submit anything containing someone else's private data, and call it out kindly if you spot it.

## Not welcome

- Harassment, personal attacks, or bad faith.
- Submitting secrets, credentials, or private data, whether yours or anyone else's.
- Using the community namespace to advertise, recruit, or spam.

## If something goes wrong

If you see something that breaks this, flag it to a maintainer privately or open an issue. We sort it out quickly and quietly, the way a good peer group does.

That is it. Keep it founder-to-founder and everyone wins.
